# Vení a Correr 2026 — Opción de diseño 2

Segunda propuesta visual para presentar al cliente.

## Concepto
Estética deportiva editorial, más enérgica y modular:
- crema, turquesa, coral y amarillo;
- tipografía grande;
- tarjetas con bordes y sombras gráficas;
- información reorganizada para evitar repeticiones;
- CTA de WhatsApp con ícono;
- logo y favicon transparentes;
- sin botones de inscripción.

## Publicación
Subí el contenido de esta carpeta a la raíz del repositorio y activá GitHub Pages desde `main / root`.
