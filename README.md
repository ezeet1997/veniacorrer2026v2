# Vení a Correr 2026 — Estilo oficial

Versión alineada con la identidad visual del evento:
- turquesa, blanco y degradados suaves;
- diseño limpio y deportivo;
- responsive para celular, tablet y escritorio;
- logo y favicon transparentes;
- botón de WhatsApp con ícono;
- sección de resultados históricos 2023, 2024 y 2025;
